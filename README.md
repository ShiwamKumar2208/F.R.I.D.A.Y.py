# F.R.I.D.A.Y.py 🤖

A simple voice assistant built in Python — inspired by the idea of having a personal AI companion that can listen, respond, and perform basic tasks.

> ⚠️ This project is currently under refactoring and modernization.

---

## 🚀 Current Status

This repository is being actively reworked to:

* Improve cross-platform compatibility (Linux + Windows)
* Refactor code into a modular structure
* Remove hardcoded paths and system dependencies
* Enhance stability and usability

---

## ✨ Features (Legacy)

* 🎤 Voice input using speech recognition
* 🔊 Text-to-speech responses
* 🌐 Open websites like YouTube, Twitter
* 🕒 Tell current time
* 🔍 Perform basic Google searches
* 📁 Launch local applications

---

## 🛠️ Tech Stack

* Python
* `pyttsx3` (Text-to-Speech)
* `SpeechRecognition`
* `wikipedia`
* Standard Python libraries

---

## ⚙️ Setup (Work in Progress)

Clone the repository:

```bash
git clone https://github.com/ShiwamKumar2208/F.R.I.D.A.Y.py.git
cd F.R.I.D.A.Y.py
```

Install dependencies:

```bash
pip install pyttsx3 SpeechRecognition wikipedia
```

> Additional system dependencies may be required depending on your OS.
